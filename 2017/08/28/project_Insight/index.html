你我都是“洞察计划”的一部分——从《美国队长2》看大数据

发表于 2017-08-28

看了《美国队长2》，标准的好莱坞特工电影：团队出现内奸叛变，英雄蒙冤出逃，一路披荆斩棘查询真相，最终众人合力查获真凶，挫败阴谋。剧情、特效、人物，我不能一一评价，网上影评很多也很专业，每个观众的兴奋点可能不一样，超级英雄、魔鬼身材、科幻装备、激情打斗，都很赞啊！电影散场，最让我印象深刻的是剧情里的“洞察计划”。

所谓的“洞察计划”，就是通过分析每个人的经历和资料，预测他未来会不会对稳定秩序造成威胁；有威胁的在威胁发生前就用空天母舰消灭了。详细点讲，就是通过一个人的过去行为，包括说过的话，看过的书，听过的音乐，看过的电影，参加的社团，发表、点赞、转发的微博，发送的电子邮件，每一通电话、每一天短信，活动圈子，职业属性，教育经历，医疗情况，血型星座等，来对这个人的未来进行预测，如果预测结果不是设计者期望的样子，那么这些人就会被“清洗”，来保证设计者的意志得到贯彻执行；在每一个对设计者不利的行为出现前，就消灭掉行为人。

看到这里，大家是不是觉得与现代法律精神——当事人为他的行为负责不符，尽管这是在电影里虚构的，但是大数据的应用，让我觉得一切似乎不是没有可能。

由于职业的关系，前两年开始接触“大数据”概念，现在应用的地方越来越多，乃至于我现在在做产品的时候，也会引用该思想，注重数据的采集和分析。大数据时代，我们的“数据”正在被很多人窥视。
斯诺登曝光的美国“棱镜计划”，就是活生生的例子，事实证明，已经有人在做这样的例子。不仅政府，包括很多企业，也有大数据应用的例子，亚马逊就是其中一个。当你在某购物网站上看到下方有“猜你喜欢”、“与你相关”等推送栏目时，请小心了，你的数据已经被窥视了。

大数据是一种思维方式，也是一种趋势，我没有办法判断它的好坏，但是在我看了维克托教授所著的《Big Date》之后，我做了一个梦：

很多年后的一天，我从睡梦中醒来，各项身体数据已自动上传到服务器，根据数据变化提醒我有感冒的可能，要多加注意；机器管家已经根据我的身体状况和口味喜好准备好了早餐，出门时机器管家已经根据我的工作安排通知了出租车在楼下，并且告知了我要去的目的地；合作伙伴已经通过我的服务器共享的工作计划知道我要来，并准备了洽谈事宜，而我根据系统预测的结果：对方极可能同意我们的合作条件准备好了合同……

然后就是我被吓醒，因为我发现，我的生活被“设计”了。当有一天，大数据预测已经可以非常准确的判断将要发生的事情，人的定义可能会被改写。一切都只是数据。所有的情感也可以被分析和预判，可能到了那一天，“我爱你”不再是“死生契阔，与子成说；执子之手，与子偕老”，而是“你的数据匹配我的数据”。
